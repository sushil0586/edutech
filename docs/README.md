# Documentation Index

This repository now keeps root-level product and planning documents organized under `docs/`.

## Folders

- `docs/reference`
  Current source-of-truth and canonical contract/reference documents.

- `docs/implementation-plans`
  Active execution plans, phased tickets, rollout plans, and implementation backlogs.

- `docs/qa-runbooks`
  QA checklists, pilot notes, staging/deployment runbooks, and operational confidence docs.

- `docs/role-maps`
  Role-wise functionality maps and Playwright coverage references.

- `docs/content-question-bank`
  Question-bank design, curriculum/content maps, and content seeding references.

- `docs/architecture-product`
  Product architecture, platform strategy, security model, and high-level design notes.

- `docs/frontend-mobile`
  Frontend, UX, mobile, and role-specific screen/spec documents.

- `docs/user-manual`
  End-user manuals and guided workflow instructions.

## Notes

- App-specific documentation inside `edutech_backend/`, `edutech_web/`, `edutech_frontend/`, and `nexora_student_mobile/` remains in place.
- Legacy duplicate docs from the nested `edutech/` mirror were removed so there is now one canonical documentation set.
