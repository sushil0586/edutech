## Question 1
Question Type: Direct Concept
Difficulty Level: Foundation
Question Text: Which of the following is a simple equation?
Options:
- A. `3x + 5`
- B. `4 + 7 = 11`
- C. `x + 8 = 13`
- D. `2a - 3b`
Correct Answer: C
Explanation: A simple equation contains a variable and an equality sign. `x + 8 = 13` satisfies both conditions. `3x + 5` and `2a - 3b` are only expressions, while `4 + 7 = 11` is a numerical statement with no variable. Other options are incorrect because they miss either the variable or the equation structure.
Common Mistake: Students often call any algebraic expression an equation even when there is no `=` sign.
Learning Objective: Identify what makes a mathematical statement a simple equation.

## Question 2
Question Type: Direct Concept
Difficulty Level: Foundation
Question Text: In the equation `p - 4 = 9`, which value of `p` makes the equation true?
Options:
- A. 5
- B. 9
- C. 13
- D. 36
Correct Answer: C
Explanation: To find `p`, add 4 to both sides: `p = 13`. If `p = 13`, then `13 - 4 = 9`, which is correct. The other options do not satisfy the equation when substituted.
Common Mistake: Students may subtract 4 from 9 instead of undoing the subtraction.
Learning Objective: Solve a one-step subtraction equation.

## Question 37

Explanation: Solving the equation gives `x = 6`. Only option C satisfies the equation. A, B, and D make the left side 8, 9, and 14 respectively. Since this is a multi-select item, the correct set contains only C.
Common Mistake: Marking both 4 and 6 because both numbers appear in the equation.

## Question 49
Question Type: Multi-Select
Difficulty Level: Advanced
Question Text: Which equations have the solution `x = 3`? Select all that apply.
Options:
- A. `x + 2 = 5`
- B. `2x = 8`
- C. `x - 1 = 2`
- D. `x/3 = 1`
Correct Answer: A, C, D
Explanation: For `x = 3`, equation A gives `3 + 2 = 5`, equation C gives `3 - 1 = 2`, and equation D gives `3/3 = 1`, all true. Equation B gives `2(3) = 6`, not 8. The main idea is to substitute carefully into each equation.
Common Mistake: Solving one option and assuming the rest are similar.
Learning Objective: Test a given value across multiple equations.

## Question 50
Question Type: Short Answer
Difficulty Level: Advanced
Question Text: The difference between three times a number and 8 is 19. Find the number.
Correct Answer: 9
Explanation: Let the number be `x`. “Three times a number” is `3x`, and “the difference between three times a number and 8 is 19” gives `3x - 8 = 19`. Add 8 to both sides: `3x = 27`. Divide by 3 to get `x = 9`. A common misconception is to write `3x + 8 = 19`, but “difference” tells us to subtract.
Common Mistake: Using addition instead of subtraction when translating “difference”.
Learning Objective: Form and solve a two-step equation from a verbal statement.
