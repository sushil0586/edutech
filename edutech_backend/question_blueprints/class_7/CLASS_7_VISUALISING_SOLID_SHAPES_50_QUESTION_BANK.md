## Question 1
Question Type: Direct Concept
Difficulty Level: Foundation
Question Text: Which of the following is a solid shape?
Options:
- A. square
- B. triangle
- C. cube
- D. line segment
Correct Answer: C
Explanation: A cube is a three-dimensional solid shape. A square and a triangle are flat plane figures, and a line segment is one-dimensional.
Common Mistake: Treating every geometric figure as a solid.
Learning Objective: Distinguish solid shapes from plane figures.

## Question 3
