# Topic 1: Indian and International Number Systems

## Benchmark set size

- 10 questions
- 2 foundation
- 3 intermediate
- 5 advanced

## Questions

### 1. Foundation

- Question type: MCQ Single
- Archetype: direct concept check
- Question:
  In the Indian system of numeration, which is the correct way to write `84572619` with commas?
- Options:
  - A. `84,57,26,19`
  - B. `8,45,72,619`
  - C. `84,572,619`
  - D. `8,45,726,19`
- Correct answer: `B`
- Explanation:
  In the Indian system, we first place a comma after the last three digits from the right. After that, commas are placed after every two digits. So `84572619` is correctly written as `8,45,72,619`.

### 2. Foundation

- Question type: MCQ Single
- Archetype: numeral to words
- Question:
  A museum recorded `12,304,508` visitors in the International system. Which reading is correct?
- Options:
  - A. Twelve lakh thirty-four thousand five hundred eight
  - B. One hundred twenty-three lakh four thousand five hundred eight
  - C. Twelve million three hundred four thousand five hundred eight
  - D. One crore twenty-three lakh four thousand five hundred eight
- Correct answer: `C`
- Explanation:
  In the International system, the number is read in groups of three digits: millions, thousands, and ones. So `12,304,508` becomes twelve million, three hundred four thousand, five hundred eight.

### 3. Intermediate

- Question type: MCQ Single
- Archetype: error detection
- Question:
  A student wrote the number name of `7,08,04,015` as “seven crore eight lakh four thousand fifteen.” What is the mistake?
- Options:
  - A. The student should have said “seven crore eighty lakh...”
  - B. The student should have said “seven crore eight lakh four thousand and fifteen hundred.”
  - C. The student should have said “seven crore eight lakh four lakh fifteen.”
  - D. There is no mistake.
- Correct answer: `D`
- Explanation:
  The number has 7 crore, 8 lakh, 4 thousand, and 15 ones. So “seven crore eight lakh four thousand fifteen” is already the correct reading, which means there is no mistake.

### 4. Intermediate

- Question type: MCQ Single
- Archetype: compare and choose
- Question:
  Which statement is correct?
- Options:
  - A. `4,09,75,210` is greater than `40,875,210`
  - B. `40,875,210` is greater than `4,09,75,210`
  - C. Both numbers are equal
  - D. The numbers cannot be compared because they are written in different systems
- Correct answer: `A`
- Explanation:
  `4,09,75,210` in the Indian system means 4 crore 9 lakh 75 thousand 210. In International notation, that is `40,975,210`. Since `40,975,210` is greater than `40,875,210`, option `A` is correct.

### 5. Intermediate

- Question type: MCQ Single
- Archetype: words to figures
- Question:
  Which numeral matches “thirty-two million forty thousand nineteen”?
- Options:
  - A. `32,040,019`
  - B. `3,20,40,019`
  - C. `32,400,019`
  - D. `32,004,019`
- Correct answer: `A`
- Explanation:
  Split the number name into parts: thirty-two million = `32,000,000`, forty thousand = `40,000`, and nineteen = `19`. Adding these parts gives `32,040,019`, so option `A` is correct.

### 6. Advanced

- Question type: MCQ Single
- Archetype: misconception correction
- Question:
  Riya says, “If two numbers have the same number of digits, then the commas placed in them must also be the same, no matter which numeration system is used.” Which is the best correction?
- Options:
  - A. Riya is correct because commas only help in reading and do not depend on the system
  - B. Riya is correct only for numbers less than one thousand
  - C. Riya is incorrect because Indian and International systems group digits differently
  - D. Riya is incorrect because commas are used only in the Indian system
- Correct answer: `C`
- Explanation:
  The number of digits may be the same, but the grouping rule can still be different. The Indian system follows a `3,2,2,...` pattern, while the International system follows a `3,3,3,...` pattern. So Riya’s statement is incorrect.

### 7. Advanced

- Question type: Short Answer
- Archetype: place-value reasoning
- Question:
  In the number `6,43,08,512`, what is the place value of the digit `3`?
- Correct answer:
  `3,00,000`
- Explanation:
  Read the number as `6 crore 43 lakh 8 thousand 512`. The digit `3` is in the lakh place within `43 lakh`, so its place value is `3 lakh`, which is written as `3,00,000`.

### 8. Advanced

- Question type: MCQ Single
- Archetype: case-based interpretation
- Question:
  A state report says: “Total saplings planted this year: `18,507,250`.” Which representation shows the same number in the Indian system?
- Options:
  - A. `1,85,07,250`
  - B. `18,50,72,50`
  - C. `18,507,250`
  - D. `185,07,250`
- Correct answer: `A`
- Explanation:
  The quantity stays the same when we switch between the two numeration systems; only the comma grouping changes. So `18,507,250` in the International system is written as `1,85,07,250` in the Indian system.

### 9. Advanced

- Question type: MCQ Single
- Archetype: assertion-reason
- Question:
  Assertion (A): Writing large numbers with commas makes them easier to read correctly.

  Reason (R): Commas help separate digits into meaningful groups according to a numeration system.

  Choose the correct option.
- Options:
  - A. Both A and R are true, and R correctly explains A
  - B. Both A and R are true, but R does not correctly explain A
  - C. A is true, but R is false
  - D. A is false, but R is true
- Correct answer: `A`
- Explanation:
  The assertion is true because commas help us read large numbers more easily. The reason is also true because commas separate digits into meaningful groups based on the numeration system. That is exactly why the number becomes easier to interpret.

### 10. Advanced

- Question type: MCQ Single
- Archetype: best explanation / elimination
- Question:
  Which option best explains why the representation `53,412,76` is not acceptable in the International system?
- Options:
  - A. The number has too many digits for the International system
  - B. The digits should always be grouped in pairs in the International system
  - C. After the first comma, the International system groups digits in threes, so this placement breaks the pattern
  - D. The International system does not use commas for 8-digit numbers
- Correct answer: `C`
- Explanation:
  In the International system, commas are placed after every three digits from the right. The form `53,412,76` breaks that pattern, so its grouping is not standard. That is why option `C` is the best explanation.

# Topic 2: Acidic, Basic, and Neutral Substances

## Benchmark set size

- 10 questions
- 2 foundation
- 3 intermediate
- 5 advanced

## Questions

### 1. Foundation

- Question type: MCQ Single
- Archetype: classification
- Question:
  Which of the following is most likely to be acidic?
- Options:
  - A. Soap solution
  - B. Lemon juice
  - C. Baking soda solution
  - D. Toothpaste
- Correct answer: `B`
- Explanation:
  Lemon juice contains citric acid, which gives it acidic properties. The other options are commonly basic substances used for cleaning or oral care. So lemon juice is the best acidic example here.

### 2. Foundation

- Question type: MCQ Single
- Archetype: indicator result
- Question:
  Blue litmus paper is dipped into a liquid and turns red. What can you conclude?
- Options:
  - A. The liquid is basic
  - B. The liquid is neutral
  - C. The liquid is acidic
  - D. The liquid is always safe to touch
- Correct answer: `C`
- Explanation:
  Blue litmus turning red is a standard sign of an acid. A base would turn red litmus blue, while a neutral substance usually would not change either paper. So the liquid is acidic.

### 3. Intermediate

- Question type: MCQ Single
- Archetype: observation-based inference
- Question:
  A student tests an unknown liquid. Red litmus remains red, but blue litmus turns red. What is the best conclusion?
- Options:
  - A. The liquid is acidic
  - B. The liquid is basic
  - C. The liquid is neutral
  - D. The liquid is a metal solution
- Correct answer: `A`
- Explanation:
  This pattern matches the behavior of an acid: blue litmus changes to red, while red litmus shows no new change. So the observations point to an acidic liquid, making option `A` correct.

### 4. Intermediate

- Question type: MCQ Single
- Archetype: daily-life application
- Question:
  Which item would most likely be used to clean a greasy kitchen surface because of its basic nature?
- Options:
  - A. Vinegar
  - B. Lemon water
  - C. Soap solution
  - D. Fruit juice
- Correct answer: `C`
- Explanation:
  Soap solution is basic and is commonly used to remove greasy or oily dirt. The acidic options like vinegar or lemon water are not the best answer for this question. So option `C` is the most suitable choice.

### 5. Intermediate

- Question type: MCQ Single
- Archetype: misconception correction
- Question:
  Aarav says, “Every acidic substance is dangerous and must never be touched or eaten.” Which is the best correction?
- Options:
  - A. Aarav is correct because all acids burn skin immediately
  - B. Aarav is incorrect because some weak acids are found in foods, though not all acids are safe in every form
  - C. Aarav is incorrect because acids and bases are the same
  - D. Aarav is correct because only bases can be used in daily life
- Correct answer: `B`
- Explanation:
  Aarav’s idea is too broad. Some acidic substances such as lemon juice and curd are safe to eat, but that does not mean all acids are harmless. So option `B` gives the best balanced correction.

### 6. Advanced

- Question type: MCQ Single
- Archetype: experiment interpretation
- Question:
  Three drops of china rose indicator are added to a liquid. The liquid turns dark pink. Which is the most reasonable conclusion?
- Options:
  - A. The liquid is acidic
  - B. The liquid is basic
  - C. The liquid must be neutral
  - D. No conclusion can be made from an indicator
- Correct answer: `A`
- Explanation:
  China rose indicator changes to a dark pink or magenta shade in acidic substances. So the observed color change suggests that the liquid is acidic. That makes option `A` the strongest conclusion.

### 7. Advanced

- Question type: MCQ Single
- Archetype: neutralization reasoning
- Question:
  A child is stung by an ant, and a mild basic paste is suggested for relief. Why can this help?
- Options:
  - A. The base may help reduce the effect of the acidic sting by partial neutralization
  - B. Bases always cool the skin instantly, regardless of the sting
  - C. Bases turn every harmful substance into water
  - D. The base makes the skin acidic enough to resist pain
- Correct answer: `A`
- Explanation:
  At this level, ant stings are often described as acidic in nature. A mild base may help reduce their effect by partial neutralization. So option `A` connects the idea correctly without making an exaggerated claim.

### 8. Advanced

- Question type: MCQ Single
- Archetype: safety reasoning
- Question:
  A bottle in the lab has no label. A student wants to taste one drop to identify whether it is acidic or basic. What is the safest response?
- Options:
  - A. Taste only a very tiny amount
  - B. Smell it closely first, then taste it
  - C. Test it with a suitable indicator instead of tasting it
  - D. Add it directly to another chemical and watch what happens
- Correct answer: `C`
- Explanation:
  Unknown substances should never be tasted because they may be harmful. A safer scientific method is to test them with a suitable indicator or by following lab instructions. That is why option `C` is the correct response.

### 9. Advanced

- Question type: MCQ Single
- Archetype: assertion-reason
- Question:
  Assertion (A): A neutral substance does not change the color of either red litmus or blue litmus.

  Reason (R): Neutral substances are neither acidic nor basic.

  Choose the correct option.
- Options:
  - A. Both A and R are true, and R correctly explains A
  - B. Both A and R are true, but R does not correctly explain A
  - C. A is true, but R is false
  - D. A is false, but R is true
- Correct answer: `A`
- Explanation:
  A neutral substance is neither acidic nor basic. Because of that, it usually does not change red litmus or blue litmus. So both the assertion and the reason are true, and the reason explains the assertion.

### 10. Advanced

- Question type: MCQ Single
- Archetype: case-based best explanation
- Question:
  A student tests two liquids. Liquid P turns turmeric paper reddish-brown. Liquid Q does not change turmeric paper at all. Which explanation is best?
- Options:
  - A. P is likely basic, while Q may be acidic or neutral
  - B. P is acidic, while Q must be basic
  - C. P is neutral, while Q must be acidic
  - D. Both liquids are definitely neutral
- Correct answer: `A`
- Explanation:
  Turmeric turns reddish-brown in a basic substance, so Liquid P is likely basic. If turmeric shows no change for Liquid Q, that result alone does not prove it is acidic; it may be acidic or neutral. So option `A` is the best explanation.
