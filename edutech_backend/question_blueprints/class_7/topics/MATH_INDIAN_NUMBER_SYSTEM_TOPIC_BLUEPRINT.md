# Indian and International Number Systems Topic Blueprint

## Topic Identity

- Class: 7
- Subject: Math
- Topic: Indian and International Number Systems
- Board / Curriculum: CBSE / premier-board compatible
- Intended pack size: 100 questions
- Recommended difficulty mix: 20 foundation, 30 intermediate, 50 advanced

## Why This Topic Needs A Separate Blueprint

This topic becomes repetitive very quickly if generation keeps asking:

- write a number in words
- write a number in figures
- place commas correctly
- identify place value

Those are necessary, but if overused they create low-value duplicates. A premium pack must move beyond basic conversion drills into comparison, error detection, reasoning, representation, and structured interpretation.

## Topic Scope

Include:

- Indian place-value chart reading
- International place-value chart reading
- comma placement in both systems
- conversion between numeral and word form
- comparison of large numbers
- expanded form
- place value and face value
- identifying errors in grouped numbers
- interpreting large values in real contexts

Exclude:

- irrational or irrational-style notation
- scientific notation
- advanced exponents
- decimal place-value topics unless explicitly combined elsewhere

## Stem Variety Bank

Allowed opening families:

- The number shown on a notice board is...
- A student grouped the digits of a number as...
- Which representation is correct?
- Study the place-value chart and answer.
- A district report mentions...
- Which mistake has been made in writing...
- Arrange the following from greatest to smallest.
- Which statement about the digit... is correct?

Rule:

- no visible opening family should appear more than twice in a row
- do not repeat “Write the following number in words” as a dominant pattern

## Context Bank

Use mixed contexts such as:

- school enrollment counts
- city populations
- sports spectators
- rainfall or distance figures
- online views or downloads
- library book counts
- government or festival budgets

## Anti-Repetition Rules

Hard bans:

- do not generate 10 versions of “insert commas in this number”
- do not generate 10 versions of “write in words” with only changed digits
- do not repeat one place-value question by changing the highlighted digit only
- do not treat bigger numbers as advanced unless reasoning also changes

Near-duplicate test:

- if two questions differ only by digit replacement while testing the same action in the same wording, rewrite one

## Difficulty Design

### Foundation

- test recognition, reading, basic conversion, comma placement
- avoid overly mechanical worksheet feel
- include at least some “spot the correct option” and chart-reading items

### Intermediate

- test conversion plus comparison, expanded form, and error detection
- avoid pure drill repetition
- include ordering, comparison, and miswritten-number correction

### Advanced

- test inference, elimination, interpretation, and multi-step reasoning
- avoid calling a long number “advanced” unless the logic is advanced
- include case-based data interpretation, assertion-reason, and subtle representation errors

## Distractor Design Rules

- misplaced commas
- confusion between Indian and International grouping
- wrong place-value interpretation
- digit-order reversal
- incorrect word form based on local grouping confusion

## Banned Weak Patterns

- “What is the product/sum/value” type patterns that do not belong to this topic
- repeated four-option conversion drills with no reasoning twist
- visible prefixes, IDs, or seed labels in the question text

## Review Checklist

- at least 15 visibly distinct stems in a 100-question pack
- at least 8 real-context questions
- both Indian and International systems are balanced
- no more than 2 near-identical comma-placement items per 25-question block
- advanced questions are not just longer numbers
