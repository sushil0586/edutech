# Debugging and Problem Solving

- Class: 7
- Subject: Computer
- Topic code: `COMP-CODING-DEBUG`

## Scope

- spotting mistakes, fixing sequence problems, and logical troubleshooting

## Premium Question Lanes

- find the bug
- choose the best fix
- identify why a result is wrong
- compare multiple possible corrections

## Anti-Repetition Rules

- do not create only “spelling error in code” items
- vary logic, order, missing-step, and wrong-condition cases
- advanced should require reasoning between fixes
- include non-code problem-solving patterns too
