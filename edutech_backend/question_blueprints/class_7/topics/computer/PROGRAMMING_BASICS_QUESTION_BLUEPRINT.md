# Programming Basics

- Class: 7
- Subject: Computer
- Topic code: `COMP-CODING-BASICS`

## Scope

- commands, sequence, simple variables or blocks, and program behavior at introductory level

## Premium Question Lanes

- predict simple program output / action
- choose correct command order
- identify purpose of a block or instruction
- compare two simple logic structures

## Anti-Repetition Rules

- do not create only keyword recall
- use block-thinking and text-command interpretations
- advanced should involve behavior reasoning
- vary movement, drawing, input, and repeat contexts

## Incremental Build Plan

- benchmark 10 with command and outcome mix
- expand 25 with simple debugging
- scale after pattern audit
