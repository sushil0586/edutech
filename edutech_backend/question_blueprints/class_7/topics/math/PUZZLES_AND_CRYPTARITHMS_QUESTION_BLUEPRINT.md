# Puzzles and Cryptarithms

- Class: 7
- Subject: Math
- Topic code: `MATH-LOGIC-PUZZLES`

## Scope

- cryptarithms, grid puzzles, and structured reasoning challenges
- letter-digit mapping at age-appropriate level
- avoid overcomplicated olympiad notation

## Premium Question Lanes

- digit-letter substitution
- find the only valid arrangement
- eliminate impossible assignments
- explain why a choice cannot work

## Anti-Repetition Rules

- do not generate only one cryptarithm shell repeatedly
- vary table, grid, sum, and assignment formats
- advanced must reward logic, not long trial-and-error only
- keep explanations clear and concise

## Incremental Build Plan

- benchmark 10 with 4+ puzzle families
- expand 25 after reviewing uniqueness manually
- scale only if duplicates stay visibly low
