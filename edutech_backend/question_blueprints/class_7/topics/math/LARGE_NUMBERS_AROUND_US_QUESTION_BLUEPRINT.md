# Large Numbers Around Us

- Class: 7
- Subject: Math
- Topic code: `MATH-NUMBERS-LARGE`

## Scope

- use large quantities from population, distance, finance, records, and counts
- focus on reading, interpreting, estimating, and comparing large numbers
- avoid reducing the topic to only comma placement drills

## Premium Question Lanes

- real-world data interpretation
- correct reading of large numerals
- ordering and nearest-estimate reasoning
- spotting unreasonable claims based on quantity size

## Anti-Repetition Rules

- do not create 10 similar “write in words” questions
- do not use only city-population examples
- do not make advanced difficulty equal to bigger digits only
- rotate contexts: tickets, books, people, money, rainfall, downloads

## Incremental Build Plan

- benchmark 10: reading, comparison, one case-based set
- expand 25: add data interpretation and error detection
- expand 50+: add richer estimation and multi-step comparison
