# Patterns and Rules

- Class: 7
- Subject: Math
- Topic code: `MATH-ALGEBRA-PATTERNS`

## Scope

- numeric and visual patterns, rule finding, and next-term reasoning
- link patterns to simple generalization
- avoid turning all questions into one sequence-completion format

## Premium Question Lanes

- infer rule from terms
- choose matching pattern representation
- identify wrong continuation
- short-answer rule explanation

## Anti-Repetition Rules

- do not create 20 “find next term” clones
- vary arithmetic, geometric-looking, alternating, and visual patterns
- advanced should ask for rule logic, not just next number
- include “which does not fit the rule” style questions

## Incremental Build Plan

- benchmark 10 with at least 5 pattern families
- expand 25 with explanation-based items
- scale after reviewing visible rule diversity
