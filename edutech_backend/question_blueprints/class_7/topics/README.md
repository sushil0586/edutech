# Class 7 Topic Blueprints

This folder contains topic-level blueprints for Class 7.

These files are more specific than family blueprints. Use them when a topic has a high risk of repeated questions or when the topic is commercially important.

Use a topic blueprint when you want to:

- stop repeated stems inside one topic pack
- cover all major sub-concepts cleanly
- guide premium manual content generation
- convert one strong topic into reusable seed logic later

Practical rule:

1. use family blueprints to improve a generator family
2. use topic blueprints to prevent repetition inside one specific topic
3. once a topic blueprint is stable, reuse it for future boards or institutes with minor adaptation

Incremental authoring rule:

1. start from the exact topic file
2. generate only a 10-question benchmark batch
3. audit repeats and weak stems
4. expand to 25
5. only then scale toward 50 or 100
