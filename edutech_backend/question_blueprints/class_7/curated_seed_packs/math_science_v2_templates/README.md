# Class 7 Math/Science 25-Question Template Packs

This folder contains authoring templates generated by:

```bash
python manage.py generate_curated_topic_pack_templates
```

If you want a deliberately different second-run scaffold, use:

```bash
python manage.py generate_curated_topic_pack_variants --variant b
python manage.py generate_curated_topic_pack_variants --variant c
```

## Purpose

These files are not final seed packs.

They are template packs that create:

- 25 differentiated question slots
- subject/topic-specific archetype spread
- difficulty distribution
- authoring prompts and review metadata

## Important Rule

Do not seed these files directly into DB.

These files are for:

1. content writing
2. content review
3. later conversion into final curated packs under:

- `question_blueprints/class_7/curated_seed_packs/math_science_v2/`

## Typical Workflow

1. Run the template generator for all Math/Science topics.
2. Open one topic JSON file.
3. Replace `[AUTHORING REQUIRED] ...` content with real reviewed questions.
4. Move the final topic file into the live curated pack folder.
5. Run `seed_curated_math_science_questions` for only those approved topic codes.

## Variant Purpose

The variant command is useful when:

- the second run should not repeat the same template order
- you want a different archetype arrangement for the same topic
- you want separate `a / b / c` authoring starts before final curation
