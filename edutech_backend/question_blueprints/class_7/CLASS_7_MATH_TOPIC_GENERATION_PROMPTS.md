# Class 7 Math Topic Generation Prompts

Use this file to generate production-ready Class 7 Math question banks with the same quality bar used for `Variables and Expressions`.

Each prompt below is designed for:

- `50` questions
- `20 foundation`
- `20 intermediate`
- `10 advanced`
- strong explanation quality
- non-repetitive question design
- NCERT-first progression with Olympiad/NSTSE-style stretch at the advanced level

## How To Use

1. Copy one topic prompt block.
2. Use it with your content-generation workflow.
3. Review the output against:
   - [CLASS_7_MATH_QUALITY_CHECKLIST.md](./curated_authoring/math_science_v2/CLASS_7_MATH_QUALITY_CHECKLIST.md)
4. If the topic maps to an existing repo topic, move the approved content into the corresponding curated Markdown pack.
5. If the topic does not yet map to the repo preset, keep it as a standalone reviewed bank first.
