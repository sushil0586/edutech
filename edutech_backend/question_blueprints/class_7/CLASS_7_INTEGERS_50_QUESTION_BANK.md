## Question 1
Question Type: Direct Concept
Difficulty Level: Foundation
Question Text: Which of the following is an integer?
Options:
- A. `3/5`
- B. `-7`
- C. `4.2`
- D. `sqrt(2)`
Correct Answer: B
Explanation: Integers are whole numbers and their negatives, including zero. `-7` is an integer. `3/5` is a fraction, `4.2` is a decimal, and `sqrt(2)` is not a whole number. Other options are incorrect because they are not whole-number values.
Common Mistake: Students often think every negative number is “different” from whole numbers and forget negatives can be integers.
Learning Objective: Recognize integers from a mixed list of numbers.

## Question 2

Difficulty Level: Foundation
Question Text: Which integer is smaller?
Options:
- A. `5`
- B. `0`
- C. `-3`
- D. `2`
Correct Answer: C
Explanation: On the number line, numbers to the left are smaller. `-3` lies to the left of 0, 2, and 5, so it is the smallest. Other options are greater because they lie to the right of `-3`.
Common Mistake: Comparing only the digit and ignoring the negative sign.
Learning Objective: Compare positive, negative, and zero on a number line.

## Question 3

- B. `-9`
- C. `1/9`
- D. `0`

Explanation: The additive inverse of a number is the number that adds with it to give zero. `9 + (-9) = 0`, so `-9` is the additive inverse. The other options do not make the sum zero.

Learning Objective: Identify additive inverses of integers.

- D. `-27`
Correct Answer: B
Explanation: `(-4) x 3 = -12`, so the expression becomes `20 - (-12) + (-5) = 20 + 12 - 5 = 27`. Other options come from subtracting `-12` incorrectly.
Common Mistake: Forgetting that subtracting a negative turns into addition.
Learning Objective: Handle brackets and sign changes in multi-step integer expressions.

## Question 48
Question Type: Multi-Select
Difficulty Level: Advanced
Question Text: Which of the following statements are true? Select all that apply.
Options:
- A. `(-2) x (-7)` is positive
- B. `5 + (-5)` equals `10`
- C. `0 - 9` is negative
- D. `(-4)` is greater than `(-1)`
Correct Answer: A, C
Explanation: A is true because the product of two negatives is positive. C is true because `0 - 9 = -9`, which is negative. B is false because `5 + (-5) = 0`, and D is false because `-4` is smaller than `-1`. Each option must be checked separately.
Common Mistake: Assuming one sign rule can be applied to every integer operation.
Learning Objective: Check multiple properties of integers in one item.

## Question 49

## Question 50

Learning Objective: Model a sequence of integer changes in a real-life situation.
