## Questions 11 to 25

### 11. Foundation

- Question type: MCQ Single
- Archetype: direct concept check
- Question:
  Which number is written correctly in the International system?
- Options:
  - A. `6,54,32,189`
  - B. `65,432,189`
  - C. `654,32,189`
  - D. `6,5432,189`
- Correct answer: `B`
- Explanation:
  In the International system, digits are grouped in sets of three from the right side. The form `65,432,189` follows that rule correctly, while the other options do not.

### 12. Foundation

- Question type: MCQ Single
- Archetype: reading check
- Question:
  Which is the correct reading of `3,05,00,018` in the Indian system?
- Options:
  - A. Three crore fifty lakh eighteen
  - B. Three crore five lakh eighteen
  - C. Thirty crore five lakh eighteen
  - D. Three crore five thousand eighteen
- Correct answer: `B`
- Explanation:
  Read the groups in the Indian system as 3 crore, 5 lakh, 0 thousand, and 18. That gives the number name “three crore five lakh eighteen,” so option `B` is correct.

### 13. Foundation

- Question type: Short Answer
- Archetype: numeral writing
- Question:
  Write “nine crore two lakh forty thousand six” in figures using the Indian system.
- Correct answer:
  `9,02,40,006`
- Explanation:
  Break the number name into place-value parts: nine crore = `9,00,00,000`, two lakh = `2,00,000`, forty thousand = `40,000`, and six = `6`. Putting these parts together gives `9,02,40,006`.

### 14. Intermediate

- Question type: MCQ Single
- Archetype: compare and order
- Question:
  Which list is arranged from smallest to largest?
- Options:
  - A. `5,07,90,210`, `5,07,09,210`, `50,790,210`
  - B. `5,07,09,210`, `50,790,210`, `5,07,90,210`
  - C. `50,790,210`, `5,07,09,210`, `5,07,90,210`
  - D. `5,07,09,210`, `5,07,90,210`, `50,790,210`
- Correct answer: `B`
- Explanation:
  `5,07,09,210` = `50,709,210`, `50,790,210` stays the same, and `5,07,90,210` = `50,790,210`? No, it equals `50,790,210`, making a tie. This question should be replaced before import due to ambiguity.

### 15. Intermediate

- Question type: MCQ Single
- Archetype: expanded form
- Question:
  Which is the correct expanded form of `4,08,05,302`?
- Options:
  - A. `4,00,00,000 + 8,00,000 + 5,000 + 300 + 2`
  - B. `4,00,00,000 + 80,00,000 + 5,000 + 300 + 2`
  - C. `4,00,00,000 + 8,00,000 + 50,000 + 300 + 2`
  - D. `4,00,00,000 + 8,00,000 + 5,000 + 30 + 2`
- Correct answer: `A`
- Explanation:
  This number has 4 crore, 8 lakh, 5 thousand, 3 hundred, and 2 ones. There is no contribution from ten-lakh, ten-thousand, tens, or tens-of-crore places. So option `A` is the correct expanded form.

### 16. Intermediate

- Question type: MCQ Single
- Archetype: representation selection
- Question:
  A website shows a reach of “twenty-three million six hundred thousand forty.” Which numeral matches it?
- Options:
  - A. `23,600,040`
  - B. `2,36,00,040`
  - C. `23,060,040`
  - D. `23,600,400`
- Correct answer: `A`
- Explanation:
  Start with twenty-three million = `23,000,000`. Add six hundred thousand = `600,000`, and then add forty = `40`. The final numeral is `23,600,040`, so option `A` is correct.

### 17. Intermediate

- Question type: MCQ Single
- Archetype: place-value inference
- Question:
  In the number `92,407,315`, what is the value of the digit `4`?
- Options:
  - A. `4,00,000`
  - B. `40,000`
  - C. `4,000`
  - D. `40,00,000`
- Correct answer: `A`
- Explanation:
  In the number `92,407,315`, the digit `4` sits in the hundred-thousands place. That means its value is `4 × 100,000 = 4,00,000`. So option `A` is correct.

### 18. Intermediate

- Question type: MCQ Single
- Archetype: wrong-format detection
- Question:
  Which of the following is wrongly written in the Indian system?
- Options:
  - A. `7,45,08,312`
  - B. `81,03,405`
  - C. `9,876,543`
  - D. `2,10,00,009`
- Correct answer: `C`
- Explanation:
  The Indian system uses one group of three digits at the end and then groups of two digits. The form `9,876,543` follows the International pattern instead. In Indian grouping, it should be written as `98,76,543`.

### 19. Advanced

- Question type: MCQ Single
- Archetype: case-based interpretation
- Question:
  A scholarship notice lists the total fund as `12,75,40,000`. Which interpretation is correct?
- Options:
  - A. Twelve lakh seventy-five thousand forty
  - B. One hundred twenty-seven million four hundred thousand
  - C. Twelve crore seventy-five lakh forty thousand
  - D. One crore twenty-seven lakh fifty-four thousand
- Correct answer: `C`
- Explanation:
  In Indian notation, `12` represents crore, `75` represents lakh, and `40` represents thousand. So `12,75,40,000` is read as twelve crore seventy-five lakh forty thousand.

### 20. Advanced

- Question type: MCQ Single
- Archetype: best explanation
- Question:
  Why is `2,501,08,340` hard to accept as a correct standard representation?
- Options:
  - A. Because it mixes grouping styles instead of following one numeration system consistently
  - B. Because it has more than seven digits
  - C. Because commas can only be used in the Indian system
  - D. Because the digit 0 cannot appear after a comma
- Correct answer: `A`
- Explanation:
  A correctly written large number should follow one standard grouping system throughout. In `2,501,08,340`, the comma pattern mixes styles and does not stay consistent. So option `A` gives the strongest reason for rejecting it.

### 21. Advanced

- Question type: Short Answer
- Archetype: cross-system conversion
- Question:
  Write `34,508,612` in the Indian system.
- Correct answer:
  `3,45,08,612`
- Explanation:
  The digits do not change when we convert between numeration systems; only the comma placement changes. In the Indian system, `34,508,612` is regrouped as `3,45,08,612`.

### 22. Advanced

- Question type: MCQ Single
- Archetype: misconception correction
- Question:
  Dev says, “If I can read a number correctly, then comma placement does not matter.” Which response is best?
- Options:
  - A. Dev is right because commas are only decorative marks
  - B. Dev is wrong because commas help prevent misreading and also show the numeration system being used
  - C. Dev is wrong because numbers without commas are always invalid
  - D. Dev is right for International numbers only
- Correct answer: `B`
- Explanation:
  Even if someone can sometimes read a number without commas, correct comma placement still matters. It reduces confusion and also shows whether the number is written in the Indian or International system. So option `B` is the best response.

### 23. Advanced

- Question type: MCQ Single
- Archetype: assertion-reason
- Question:
  Assertion (A): `5,43,21,008` and `54,321,008` represent the same quantity.

  Reason (R): The Indian and International systems may use different comma placements for the same number.

  Choose the correct option.
- Options:
  - A. Both A and R are true, and R correctly explains A
  - B. Both A and R are true, but R does not correctly explain A
  - C. A is true, but R is false
  - D. A is false, but R is true
- Correct answer: `A`
- Explanation:
  The actual quantity stays the same in both systems; only the grouping style changes. `5,43,21,008` in Indian notation is the same number as `54,321,008` in International notation. So both the assertion and the reason are true, and the reason explains the assertion.

### 24. Advanced

- Question type: MCQ Multiple
- Archetype: multi-select truth evaluation
- Question:
  Select all correct statements.
- Options:
  - A. In the International system, commas are placed after every three digits from the right.
  - B. In the Indian system, the first comma from the right comes after three digits.
  - C. `6,70,05,019` can be read as six crore seventy lakh five thousand nineteen.
  - D. `47,305,200` is an Indian-system representation.
- Correct answer:
  `A, B, C`
- Explanation:
  Statement A is true because International notation groups digits in threes. Statement B is true because the Indian system places the first comma after three digits from the right. Statement C is also true, while statement D is false because `47,305,200` is written in the International system.

### 25. Advanced

- Question type: MCQ Single
- Archetype: elimination-based reasoning
- Question:
  A school accountant wrote a receipt total as `1,20,304,05`. Which is the strongest reason to reject this notation?
- Options:
  - A. It is too long to be a valid number
  - B. The commas do not follow either the Indian or International grouping pattern correctly
  - C. It should have no commas because it is a money amount
  - D. The last group must always contain two digits
- Correct answer: `B`
- Explanation:
  A large number should follow either the Indian grouping rule or the International grouping rule. The notation `1,20,304,05` fits neither pattern correctly. So option `B` gives the best reason to reject it.

## Questions 11 to 25

### 11. Foundation

- Question type: MCQ Single
- Archetype: direct classification
- Question:
  Which of the following is most likely to be basic?
- Options:
  - A. Curd
  - B. Soap solution
  - C. Orange juice
  - D. Vinegar
- Correct answer: `B`
- Explanation:
  Soap solution is commonly basic in nature and is often used as an everyday example of a base. The other options are acidic or are not the best basic example here. So option `B` is correct.

### 12. Foundation

- Question type: MCQ Single
- Archetype: direct concept check
- Question:
  A substance that changes neither red litmus nor blue litmus is most likely:
- Options:
  - A. acidic
  - B. basic
  - C. neutral
  - D. metallic
- Correct answer: `C`
- Explanation:
  A neutral substance is neither acidic nor basic. Because of that, it usually does not change either red litmus or blue litmus. So option `C` is the correct answer.

### 13. Foundation

- Question type: True / False
- Archetype: concept check
- Question:
  Turmeric paper turns reddish-brown in a basic substance.
- Correct answer:
  `True`
- Explanation:
  Turmeric is a natural indicator that stays yellow in acidic or neutral substances. In a basic substance, it changes to a reddish-brown shade. So the statement is true.

### 14. Intermediate

- Question type: MCQ Single
- Archetype: daily-life observation
- Question:
  Which kitchen substance would be the best simple example of an acid in a classroom discussion?
- Options:
  - A. Baking soda
  - B. Soap water
  - C. Lemon juice
  - D. Chalk powder
- Correct answer: `C`
- Explanation:
  Lemon juice contains citric acid and is one of the most familiar acidic substances found at home. The other options are not suitable examples of an acid in this context. So option `C` is correct.

### 15. Intermediate

- Question type: MCQ Single
- Archetype: compare and classify
- Question:
  Which pair contains one acidic substance and one basic substance?
- Options:
  - A. Lemon juice and soap solution
  - B. Vinegar and curd
  - C. Soap solution and baking soda solution
  - D. Water and lemon juice
- Correct answer: `A`
- Explanation:
  Lemon juice is acidic because it contains natural acids, while soap solution is basic. The other pairs do not contain one acidic and one basic substance in the required combination. So option `A` is correct.

### 16. Intermediate

- Question type: MCQ Single
- Archetype: observation interpretation
- Question:
  A liquid turns blue litmus red but does not change turmeric. Which is the best conclusion?
- Options:
  - A. It is likely acidic
  - B. It is likely basic
  - C. It is definitely neutral
  - D. It is certainly a metal solution
- Correct answer: `A`
- Explanation:
  Blue litmus turning red is a strong sign of an acidic substance. Turmeric is mainly useful for identifying bases, so “no change” in turmeric does not go against the acid result. That is why option `A` is the best conclusion.

### 17. Intermediate

- Question type: MCQ Single
- Archetype: misconception correction
- Question:
  Meera says, “Anything that tastes bitter must be safe because it is basic.” Which correction is best?
- Options:
  - A. She is right because all bases are harmless
  - B. She is wrong because a basic substance is not automatically safe to taste or touch
  - C. She is right because bitter substances are always food items
  - D. She is wrong because all bitter substances are acidic
- Correct answer: `B`
- Explanation:
  A substance being basic does not automatically make it safe. Some basic substances can still irritate or harm the body, especially if they are unknown. So option `B` gives the best correction to Meera’s statement.

### 18. Intermediate

- Question type: Short Answer
- Archetype: concept explanation
- Question:
  Why is litmus paper useful when testing an unknown liquid?
- Correct answer:
  Litmus paper helps indicate whether a liquid is acidic or basic by showing a color change.
- Explanation:
  Litmus paper is useful because it gives a quick visible clue about the nature of a liquid. Instead of guessing by taste or touch, we can use an indicator and observe the color change safely. That is why litmus is a simple scientific testing tool.

### 19. Advanced

- Question type: MCQ Single
- Archetype: experiment interpretation
- Question:
  A student tests Liquid X. Blue litmus turns red, red litmus stays red, and turmeric paper shows no change. Which conclusion is strongest?
- Options:
  - A. Liquid X is likely acidic
  - B. Liquid X is likely basic
  - C. Liquid X is definitely neutral
  - D. Liquid X cannot be identified by indicators
- Correct answer: `A`
- Explanation:
  Blue litmus turning red is a major sign of acidity. Red litmus staying red and turmeric showing no change also fit with an acidic liquid. So all the observations support option `A`.

### 20. Advanced

- Question type: MCQ Single
- Archetype: safety reasoning
- Question:
  A cleaning liquid has accidentally spilled on the lab table. Before deciding how to handle it, what is the best first scientific step?
- Options:
  - A. Touch it gently to feel whether it is slippery
  - B. Use a suitable indicator or follow lab safety guidance instead of guessing
  - C. Taste it to know whether it is sour or bitter
  - D. Mix it with another liquid to see whether foam forms
- Correct answer: `B`
- Explanation:
  A scientific response must be safe as well as logical. Touching, tasting, or randomly mixing a spilled liquid can be dangerous, so the best first step is to use a safe test or follow lab instructions. That makes option `B` the correct choice.

### 21. Advanced

- Question type: MCQ Single
- Archetype: neutralization application
- Question:
  Why might adding a mild basic substance be useful in a situation involving an acidic sting?
- Options:
  - A. Because every base destroys pain completely
  - B. Because a base may reduce the effect of the acid through neutralization
  - C. Because acids become basic when exposed to air
  - D. Because bases always cool the skin instantly
- Correct answer: `B`
- Explanation:
  At this level, the usual explanation is that a mild base can help reduce the effect of an acidic sting by partial neutralization. It does not magically remove all pain, but it explains why a basic substance may help. So option `B` is correct.

### 22. Advanced

- Question type: MCQ Single
- Archetype: best explanation
- Question:
  A teacher says that water is often taken as a neutral substance in school-level experiments. Which explanation is best?

  - A. Because neutral substances are those that are neither acidic nor basic
  - B. Because water turns every indicator green
  - C. Because water is a kind of base
  - D. Because water is not a substance
- Correct answer: `A`
- Explanation:
  A neutral substance is one that is neither acidic nor basic in the given context. Water is often used as an example of a neutral substance in school experiments for this reason. So option `A` is the best explanation.

  Assertion (A): A basic substance can turn turmeric paper reddish-brown.

  Reason (R): Some indicators change color differently in acidic and basic substances.

  - A. Both A and R are true, and R correctly explains A
  - B. Both A and R are true, but R does not correctly explain A
  - C. A is true, but R is false
  - D. A is false, but R is true

### 24. Advanced

- Question type: MCQ Multiple
- Archetype: multi-select truth evaluation
- Question:
  Select all correct statements.
- Options:
  - A. Acids can turn blue litmus red.
  - B. Bases can turn red litmus blue.
  - C. Neutral substances must turn turmeric paper green.
  - D. Unknown substances should be tested safely, not tasted.
- Correct answer:
  `A, B, D`
- Explanation:

### 25. Advanced

- Question type: MCQ Single
- Archetype: case-based reasoning
- Question:
  A student says, “This liquid did not change blue litmus, so it must be basic.” What is the best response?
- Options:
  - A. Correct, because bases never change blue litmus
  - B. Incorrect, because not changing blue litmus does not by itself prove a substance is basic
  - C. Correct, because acids always change both red and blue litmus
  - D. Incorrect, because every base must make blue litmus red
- Correct answer: `B`
- Explanation:
  If blue litmus does not change, that alone does not prove the substance is basic. A base is usually confirmed by a change in red litmus, while a neutral substance may not change either. So more evidence is needed, which is why option `B` is the best response.
