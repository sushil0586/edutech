# Class 7 Math and Science Benchmark Outlines

This file defines the first audited benchmark outlines for rebuilding repeated Class 7 question content.

These are not bulk seed packs.

These are pilot structures for:

- `Math`: Indian and International Number Systems
- `Science`: Acidic, Basic, and Neutral Substances

Use them to create only `10` premium questions per topic first.

## Why This File Exists

The earlier question sets became repetitive because generation scaled too early.

This file prevents that by forcing:

- small-batch validation
- archetype diversity
- sub-concept spread
- anti-repetition review before expansion

## Output Expectation

For each topic:

- create exactly `10` questions
- use the outline below
- do not add extra similar questions “for safety”
- after drafting, review all 10 together before approving

## Topic 1: Math

### Indian and International Number Systems

Reference topic file:

- [topics/math/INDIAN_AND_INTERNATIONAL_NUMBER_SYSTEMS_QUESTION_BLUEPRINT.md](./topics/math/INDIAN_AND_INTERNATIONAL_NUMBER_SYSTEMS_QUESTION_BLUEPRINT.md)

### Benchmark target

- 2 foundation
- 3 intermediate
- 5 advanced

### Required sub-concept spread

- Indian grouping
- International grouping
- words to numerals
- numerals to words
- correct comma placement
- wrong representation detection
- place-value interpretation in large numbers
- real-life large-number interpretation

### Required 10-question outline

1. `Foundation`
   - archetype: direct concept check
   - focus: identify correct Indian grouping for a large number
   - anti-repeat note: do not make this a plain comma insertion drill only

2. `Foundation`
   - archetype: direct conversion
   - focus: convert a moderately large number from numeral to words in one system
   - anti-repeat note: use a fresh stem, not “Write the following in words”

3. `Intermediate`
   - archetype: error detection
   - focus: spot the incorrectly grouped representation among options
   - anti-repeat note: distractors must reflect realistic grouping mistakes

4. `Intermediate`
   - archetype: compare and choose
   - focus: compare two large numbers written in different presentation styles
   - anti-repeat note: comparison should require structure understanding, not only digit count

5. `Intermediate`
   - archetype: words-to-figures
   - focus: translate a number name into correct numeral form
   - anti-repeat note: avoid making this a near-copy of question 2

6. `Advanced`
   - archetype: misconception correction
   - focus: correct a student mistake about Indian vs International comma grouping
   - anti-repeat note: this should test understanding, not memory only

7. `Advanced`
   - archetype: place-value reasoning
   - focus: interpret the value of a digit in a large number across the correct grouping system
   - anti-repeat note: avoid a plain highlighted-digit worksheet style

8. `Advanced`
   - archetype: case-based interpretation
   - focus: use a real-life large number from a report, population, budget, or audience count
   - anti-repeat note: ask what the number means or which representation is best

9. `Advanced`
   - archetype: assertion-reason
   - focus: relationship between number grouping and readability / interpretation
   - anti-repeat note: do not make the reason a trivial restatement

10. `Advanced`
   - archetype: best-explanation / elimination
   - focus: choose the best explanation for why one numeral representation is invalid or misleading
   - anti-repeat note: final question should feel premium, not routine

### Benchmark approval rules

Approve this 10-question set only if:

- at least 7 visibly distinct stem shapes are used
- no more than 2 are direct conversion style
- no two questions feel like number-swapped clones
- at least 3 questions require reasoning beyond recall

## Topic 2: Science

### Acidic, Basic, and Neutral Substances

Reference topic file:

- [topics/science/ACIDIC_BASIC_AND_NEUTRAL_SUBSTANCES_QUESTION_BLUEPRINT.md](./topics/science/ACIDIC_BASIC_AND_NEUTRAL_SUBSTANCES_QUESTION_BLUEPRINT.md)

### Benchmark target

- 2 foundation
- 3 intermediate
- 5 advanced

### Required sub-concept spread

- identifying acidic, basic, and neutral substances
- indicator behavior
- litmus interpretation
- observation-based inference
- neutralization idea
- safe handling
- misconception correction
- daily-life application

### Required 10-question outline

1. `Foundation`
   - archetype: classification
   - focus: identify one acidic, basic, or neutral substance correctly
   - anti-repeat note: do not use the most overused household pairings only

2. `Foundation`
   - archetype: indicator result
   - focus: predict a simple litmus or indicator change
   - anti-repeat note: keep it clean and direct, but not trivial

3. `Intermediate`
   - archetype: observation-based inference
   - focus: infer the nature of a substance from a color-change observation
   - anti-repeat note: this must not just restate question 2

4. `Intermediate`
   - archetype: daily-life application
   - focus: choose the best use or handling decision involving acidic/basic substances
   - anti-repeat note: use a practical context, not a textbook list

5. `Intermediate`
   - archetype: misconception correction
   - focus: fix a false belief such as “all acids are dangerous” or similar
   - anti-repeat note: correction should be academically meaningful

6. `Advanced`
   - archetype: experiment interpretation
   - focus: interpret a simple classroom test using more than one observation
   - anti-repeat note: ask for the best conclusion, not only a label

7. `Advanced`
   - archetype: neutralization reasoning
   - focus: identify or explain a simple neutralization situation
   - anti-repeat note: do not make it formulaic

8. `Advanced`
   - archetype: safety reasoning
   - focus: decide the safest next step with an unknown household or lab substance
   - anti-repeat note: options must feel realistic

9. `Advanced`
   - archetype: assertion-reason
   - focus: acids / bases / indicators / neutralization cause-effect
   - anti-repeat note: do not use a shallow paraphrase as the reason

10. `Advanced`
   - archetype: case-based best explanation
   - focus: choose the best explanation from a small real-world or classroom scenario
   - anti-repeat note: the final question should combine observation and reasoning

### Benchmark approval rules

Approve this 10-question set only if:

- at least 7 visibly distinct stem shapes are used
- litmus or indicator questions do not dominate the set
- at least 3 questions involve inference or decision-making
- no two questions feel like object-swapped copies

## Review Method Before Expansion

After drafting both 10-question sets, review them using this quick checklist:

1. Are there repeated opening stems?
2. Are there near-duplicate logic patterns?
3. Are the advanced questions truly advanced?
4. Are distractors believable?
5. Would these feel premium enough to sell?

If the answer to any of the above is weak, rewrite the set before moving to `25`.

## What Comes After Approval

Only after these two benchmark sets are approved:

1. create the first `25-question` version for each topic
2. run a repeat audit again
3. then decide whether to scale to `50` or `100`

Do not jump from this benchmark directly to all Math and Science topics.
