## Question 1
Question Type: Direct Concept
Difficulty Level: Foundation
Question Text: Which instrument is used mainly to draw a circle?
Options:
- A. ruler
- B. compass
- C. protractor
- D. set square
Correct Answer: B
Explanation: A compass is used to draw circles and arcs by keeping one arm fixed and rotating the other. The other tools measure or draw straight lines and angles.
Common Mistake: Choosing the ruler because circles have a radius length.
Learning Objective: Identify the correct tool for drawing circles.

## Question 2
Question Type: Direct Concept
Difficulty Level: Foundation
Question Text: Which instrument is best for measuring an angle during construction?
Options:
- A. compass
- B. divider
- C. protractor
- D. graph paper
Correct Answer: C
Explanation: A protractor measures angles in degrees. A compass is used for arcs, a divider for transferring distance, and graph paper for plotting.
Common Mistake: Confusing angle construction with angle measurement.
Learning Objective: Match a geometry task to the correct instrument.

## Question 3
Question Type: Direct Concept
Difficulty Level: Foundation

Options:
- A. it gives a starting side or base

Learning Objective: Understand the role of a base segment in construction.
